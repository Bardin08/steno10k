Eight stages. Each one a black box you can switch off.

Every stage caches to disk, so re-runs are cheap and partial runs are free. Disable any stage from config or the UI — dependencies cascade automatically.

01 / normalizeNormalizeRename source files to safe, unicode-clean names.local
02 / chunkChunkSplit long audio into overlapping ffmpeg segments.local
03 / transcribeTranscribefaster-whisper per chunk, in parallel, on CPU.local
04 / cleanCleanFix punctuation & recognition errors via LLM.llm
05 / mergeMergeConcatenate raw + clean transcripts to markdown.local
06 / summarizeSummarizeProduce summary.md from the merged transcript.llm
07 / bundleBundleRender transcript + summary to .docx.local
08 / notifyNotifyPush results to Telegram — off by default.disabled

Standing — the three-part test

gemini-3.1-flash · 892 words → 5 points
  • Injury in fact — a concrete, particularized harm; generalized grievances don't qualify.
  • Causation — the injury must be traceable to the challenged conduct.
  • Redressability — a favorable ruling must be able to remedy the harm.
