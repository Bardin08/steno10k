# steno10k documentation

steno10k turns any recording into a clean transcript and summary, entirely on
your own machine. These docs cover installing, running, and configuring it.

- **[Getting started](/steno10k/docs/getting-started/)** — run it in one container and process your first recording.
- **[Configuration](/steno10k/docs/configuration/)** — the eight pipeline stages and how to tune them.
- **[Configuration reference](/steno10k/docs/configuration-reference/)** — every `config.yaml` key, with types and defaults.
- **[Pipeline & artifacts](/steno10k/docs/pipeline/)** — the stages in depth, the on-disk layout, and caching.
- **[CLI](/steno10k/docs/cli/)** — drive projects, sets, and runs from the terminal.
- **[API reference](/steno10k/docs/api/)** — REST endpoints, the response envelope, and the run event stream.
- **[Self-hosting](/steno10k/docs/self-hosting/)** — deploy, pick a Whisper model, and size memory.
- **[Troubleshooting & FAQ](/steno10k/docs/troubleshooting/)** — memory, model download, local-only, and more.
